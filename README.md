# Workshop Management System

## Description:
Приложение для автоматизации учета ремонтов в автомастерских. Позволяет вести учет машин, мастеров, мастерских и выполненных работ. Включает систему автоматического накопления статистики по мастерам.

## Installation: 
1. Установите компилятор G++. 2. Склонируйте репозиторий. 3. Скомпилируйте: g++ main.cpp -o app.

## Usage: 
Запустите ./app. Введите данные о машине и ремонте.
